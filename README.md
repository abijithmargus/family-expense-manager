# 🌌 Family Finance Manager: Elite Edition

A cinematic, high-performance financial command center for families. Built with a "Billionaire Startup" aesthetic, this platform provides a unified views of your family's transactions, loans, and shared growth through an immersive 3D digital cosmos.

## 🚀 Key Features

- **Cinematic 3D Universe**: An immersive React Three Fiber (R3F) background that reacts to your device's physical tilt (Mobile) and mouse movement (PC).
- **Wealth Ribbon Stat Scroller**: A high-density, thumb-friendly horizontal scroller for rapid financial metrics analysis on mobile devices.
- **Elite Analytics**: Responsive, auto-scaling charts that adapt to your viewport for a "Fintech-Elite" reporting experience.
- **QuickAction FAB**: A glowing Floating Action Button for lightning-fast entry of Expenses, Savings, and Loans.
- **Unified Family Identity**: Cross-account tracking for the whole family (Abi, Angel, Appa, Amma) with secure Google OAuth 2.0 integration.

## 🛠 Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS v4, React Three Fiber (3D), Framer Motion.
- **Backend**: FastAPI (Python), SQLAlchemy (Async), PostgreSQL.
- **Tunneling**: Multi-tunnel support via Cloudflare (UI) and Ngrok (API) for seamless mobile access.
- **Process Management**: Integrated PowerShell automation for "One-Push" deployments.

## 🚦 Getting Started

### 1. Requirements
- Python 3.12+
- Node.js 18+
- PostgreSQL
- Ngrok & Cloudflare (for remote mobile access)

### 2. Quick Launch
Run the unified deployment script to wake up the 3D ecosystem:
```powershell
./deploy.ps1
```
This will:
1. Pull the latest code.
2. Build the Frontend.
3. Launch the Backend API (Ngrok Tunnel).
4. Launch the Frontend UI (Cloudflare Tunnel).
5. Automatically sync the API endpoints in your code.

### 3. Mobile Access
Once the script is running, a **Cloudflare URL** will be generated and opened in your browser. Access this URL on your phone to enter the ecosystem.
